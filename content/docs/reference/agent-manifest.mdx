# Agent Manifest Reference

This page is the authoritative reference for all fields in an AEGIS agent manifest YAML file.

---

## Annotated Full Example

```yaml
apiVersion: 100monkeys.ai/v1      # required; must be exactly this value
kind: Agent                        # required; must be exactly "Agent"

metadata:
  name: python-coder               # required; unique name within the deployment
  labels:                          # optional; arbitrary key-value pairs for filtering
    type: developer
    team: platform

spec:
  description: "Writes Python solutions to programming tasks."   # optional
  image: myregistry/my-agent:latest                              # required
  capabilities:                    # required; list of MCP tool capabilities
    - fs.read
    - fs.write
    - cmd.run
    - web.fetch

  security:                        # required
    network_policy:
      mode: allow                  # required; "allow" = use allowlist (least privileged)
      allowlist:                   # required when mode=allow
        - pypi.org
        - api.github.com

    filesystem_policy:
      read_paths:                  # required; paths inside container
        - /workspace
        - /agent
      write_paths:                 # required
        - /workspace

    security_context: default      # required; name of a SecurityContext in config

  resources:
    cpu_quota: 1.0                 # optional; fractional CPUs. Default: 1.0
    memory_bytes: 1073741824       # optional; bytes. Default: 1 GiB
    timeout_secs: 300              # optional; per-iteration timeout. Default: 300

  max_iterations: 10               # optional; max 100monkeys attempts. Default: 10

  validation:                      # optional; list of validators. Default: exit_code only
    - type: exit_code
      expected: 0
    - type: json_schema
      schema_path: /agent/schema.json
      target_path: /workspace/result.json
      min_score: 1.0
    - type: semantic
      judge_agent: code-quality-judge
      criteria: "Evaluate overall code quality and correctness."
      min_score: 0.80
      min_confidence: 0.70

  environment:                     # optional; env vars injected into container
    PYTHONUNBUFFERED: "1"
    LOG_LEVEL: "debug"

  volumes:                         # optional; list of volume mounts
    - name: workspace
      mount_path: /workspace
      access_mode: ReadWrite
      storage_class: ephemeral
      ttl: 1h
      size_limit_bytes: 5368709120

    - name: shared-data
      mount_path: /data
      access_mode: ReadOnly
      storage_class: persistent
      volume_id: "vol-a1b2c3d4-..."
```
